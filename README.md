# INNOV-XUS-Hackathon-repo
# Smart Drug Supply Chain & Shortage Prediction Platform

This repository contains the presentation and a prototype developed for **Frost Hack**, a hackathon organized by **INNOV-XUS**.

Our project focuses on building a **Smart Drug Supply Chain & Shortage Prediction Platform** that aims to improve visibility across the pharmaceutical supply chain. The idea is to help hospitals, pharmacies, and distributors monitor medicine inventory, anticipate demand, and reduce shortages through data-driven insights.

The current version of this repository includes a **prototype interface** that demonstrates the proposed workflow and system design. The code provided here is **only a front-end prototype** created to showcase the concept and user interface during the hackathon submission.

⚠️ Note:

* The prototype is **non-functional** and does not include backend logic or working predictive models.
* It is intended **only for demonstration and presentation purposes**.

This project was developed as part of the **Frost Hack Hackathon submission**.
